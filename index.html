fSphinx

fSphinx easily builds faceted search systems using Sphinx.

Download .zip Download .tar.gz View on GitHub

fSphinx adds an easy way to perform faceted search with Sphinx. With fSphinx you can compute, preload or cache facets. You can select or deselect query terms or you can easily retrieve hits from a database.

fSphinx is used in Cloud Mining: IMDb, DBLP or MEDLINE. To learn more, please feel free to follow this tutorial.

fSphinx has been ported to PHP. Thank you Chris.
